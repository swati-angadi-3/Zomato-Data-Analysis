# Zomato-Data-Analysis
Zomato Data Analysis using Python and Jupyter Notebook

## Project Overview
This project analyzes Zomato restaurant data using Python and Jupyter Notebook.

## Files
- Zomato data.xlsx → Original dataset
- zomato_cleaned.xlsx → Cleaned dataset
- ZomatoProject.ipynb → Data cleaning and analysis code

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook

## Analysis Performed
- Data Cleaning
- Handling Missing Values
- Exploratory Data Analysis
- Visualization
